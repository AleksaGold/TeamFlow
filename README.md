
### TeamFlow - Backend
Backend для веб-приложения для внутреннего управления командами и задачами внутри компании.

### О проекте

**Стек технологий:**
- Проект написан с испoльзованием версии **Python 3.12**
- **Django 5.2.1:** Основной фреймворк для разработки
- **Django REST Framework 3.16.0:** Используется для создания API
- **PostgreSQL:** Используется в качестве базы данных
- **Simple JWT (djangorestframework-simplejwt):** Используется для настройки JWT-аутентификация для DRF
- **DRF Spectacular (документация API):** Для автоматической генерации OpenAPI 3.0 схем и документации
- **python-dotenv (переменные окружения):** Для настройки переменных окружения



### Функциональность

**Пользователи:**
- Регистрация по email и паролю
- Авторизация / выход из системы
- Роли: пользователь, менеджер, админ команды
- Обновление профиля и удаление аккаунта

**Команды:**
- Создание команды (только админ)
- Управление участниками (добавление / удаление)
- Просмотр состава команды

**Задачи:**
- Создание задач (менеджером)
- Назначение исполнителей
- Статусы задач: открыто, в работе, выполнено
- Дедлайн, описание, изменение и удаление
- Упрощённый чат внутри задачи (комментарии)

**Оценка работы:**
- Оценка задач по шкале от 1 до 5 (менеджером)
- Просмотр всех полученных оценок пользователем
- Подсчёт средней оценки за период

**Встречи:**
- Назначение встречи: дата, время, участники
- Проверка пересечений по времени
- Просмотр списка встреч пользователем
- Отмена встреч

**Календарь:**
- Отображение задач и встреч по дням
- Режимы: дневной и месячный
- Автоматическая подгрузка событий

**Админ-панель:**
- Готовая административная панель Django
- Управление пользователями, командами, задачами, встречами и оценками

### Установка и настройка

#### Клонируйте репозиторий:
```
https://github.com/AleksaGold/TeamFlow
```
#### Установите зависимости:
Для запуска проекта и установки зависимостей вам необходимо создать виртуальное окружение, сделать это можно с помощью команды
```
poetry install
```
#### Настройка переменных окружения:
Создайте файл `.env` в корне проекта и заполните его переменными окружения указанными в файле `.env.sample`.

#### Настройте базу данных в settings.py и выполните миграции:
```
python manage.py migrate
```
#### Создайте суперпользователя:
```
python manage.py createsuperuser
```

#### Запустите сервер:
```
python manage.py runserver
```

### Документация API
Документация автоматически доступна по адресу:
```
http://localhost:8000/api/docs/ 
```
